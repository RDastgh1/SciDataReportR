# Calculate robust M-scores for numeric variables

Compatibility alias for
[`CreateMScoreObject()`](https://rdastgh1.github.io/SciDataReportR/reference/CreateMScoreObject.md).
Prefer
[`CreateMScoreObject()`](https://rdastgh1.github.io/SciDataReportR/reference/CreateMScoreObject.md)
in new code because this function returns reusable M-score parameters.

## Usage

``` r
CalcMScore(...)
```

## Arguments

- ...:

  Arguments passed to
  [`CreateMScoreObject()`](https://rdastgh1.github.io/SciDataReportR/reference/CreateMScoreObject.md).

## Value

The same `MScoreObj` returned by
[`CreateMScoreObject()`](https://rdastgh1.github.io/SciDataReportR/reference/CreateMScoreObject.md).
