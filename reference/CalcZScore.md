# Calculate Z-scores (or standardized scores) and return data + parameters

Compatibility alias for
[`CreateZScoreObject()`](https://rdastgh1.github.io/SciDataReportR/reference/CreateZScoreObject.md).
Prefer
[`CreateZScoreObject()`](https://rdastgh1.github.io/SciDataReportR/reference/CreateZScoreObject.md)
in new code because this function returns a reusable `ZScoreObj`.

## Usage

``` r
CalcZScore(...)
```

## Arguments

- ...:

  Arguments passed to
  [`CreateZScoreObject()`](https://rdastgh1.github.io/SciDataReportR/reference/CreateZScoreObject.md).

## Value

The same `ZScoreObj` returned by
[`CreateZScoreObject()`](https://rdastgh1.github.io/SciDataReportR/reference/CreateZScoreObject.md).
