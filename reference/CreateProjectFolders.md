# Create Project Folder Structure

This function creates a project folder structure with the following
directories:

- Data/

  - Raw/

  - Clean/

- Scripts/

- Reports/

## Usage

``` r
CreateProjectFolders(base_path = ".")
```

## Arguments

- base_path:

  A character string specifying the base directory where the folders
  should be created. Defaults to the current working directory.

## Value

A message indicating that the project structure has been created
successfully.
